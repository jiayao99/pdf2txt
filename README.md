# pdf2txt
A python-based PDF-to-TXT converter with GUI.

## How to use:

## Step 1.
```console
pip install -r requirements.txt
```


## Step 2.
```console
python3 app.py
```

## Step 3.
Drag & Drop or Browse & Select PDF files to convert. Click "Process". The converted TXT file will be save under '/Download' Directory.
